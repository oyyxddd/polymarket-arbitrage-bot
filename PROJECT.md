# Polymarket 套利机器人

## 项目目标
复刻 0x8dxd 的套利策略，把小资金滚成大资金。

## 核心策略（跨市场套利）
1. 实时监控 Binance/Coinbase 的 BTC 现货价格
2. 监控 Polymarket 的 BTC 相关预测市场
3. 发现价差 > 2% 时闪电下单
4. 单次收益上限 5%，不贪心
5. 靠量取胜，不是单次暴利

## 技术栈
- Python 3.11+（主语言，方便异步）
- Polymarket Python SDK（官方支持）
- Binance WebSocket（实时价格）
- asyncio（高频并发）

## 功能模块
1. **价格监控器** - 同时监听币安 + Polymarket
2. **套利检测器** - 实时计算价差
3. **下单执行器** - 发现机会立即下单
4. **风控管理** - 单次上限、总资金控制
5. **日志系统** - 记录所有交易

## 启动资金
用户愿意出资（具体金额待定）

## 目标收益
不一定 1400 倍，但稳定赚钱

## 风险提示
- Polymarket 可能封号
- 网络延迟可能导致套利失败
- 预测市场流动性可能不足
